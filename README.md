# LunaGurt 🧋

Aplicación web para pedidos de yogurt artesanal con flujo secuencial de compra.

## Características

- **Flujo de compra en 3 pasos**: Pedido → Sabores → Detalles
- **Catálogo de productos**: Individual ($5), Dúo ($10), Pack Familiar ($13)
- **8 sabores**: Fresa, Piña, Mora, Durazno, Parchita, Natural, Tamarindo, Coco Piña
- **Pedido por WhatsApp**: Mensaje formateado con todos los datos
- **Diseño responsive**: Mobile-first con Material Design
- **Progreso visual**: Indicador de pasos

## Tech Stack

- React 19 + Vite
- React Compiler enabled
- CSS custom

## Getting Started

```bash
npm install
npm run dev
```

## build

```bash
npm run build
npm run lint
```
